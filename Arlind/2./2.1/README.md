# Auftrag 2.1: Dienstleistungen von Cloud Anbietern vergleichen

Teilen Sie sich in 2 bis 3-er Gruppen auf und wählen Sie einen der “grösseren” Cloud Provider aus: Google Cloud, AWS, Microsoft Azure.

Erstellen Sie in der Gruppe eine Präsentation, die folgendes umfasst:

- Angaben zum Provider (Gründungsjahr, Firmensitz, Umsatz)
- Angaben zur Cloud und den Diensten (Anzahl Dienste im Angebot, Anzahl Rechenzentren auf der Welt und pro Region)
- Angaben zu den wichtigsten Services, wie heissen z.B. die folgende Dienste:
    - Einfache virtuelle Maschine (IaaS)
    - Plattform für Applikationen (PaaS)
    - ein oder mehrere Datenbankdienste
    - Funktion als Dienst (FaaS)
    - Speicherplatzdienst
- Angaben zu den Kosten:
    - Was kostet 1 Monat einer virtuellen Maschine mit 2 vCPUs und 4GB RAM ungefähr?
